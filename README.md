## 校企合作投递系统

**技术栈：** Node.js + Express + SQLite + EJS + Bootstrap  
**功能：** 学生投递简历、企业发布岗位并查看投递、管理员审核投递与查看统计。

### 安装与运行

1. 安装依赖：

```bash
npm install
```

2. 启动开发服务器（会自动初始化数据库并创建默认管理员账号）：

```bash
npm run dev
```
或生产模式：

```bash
npm start
```

下面命令会重置数据库，系统完全重置
```bash
npm run reset-db
```

3. 浏览器访问：`http://localhost:3000`

### 默认账号

- 管理员：admin / admin123
- 学生、企业请自行在“注册”页面创建账号。

### 角色功能概览

- **学生**
  - 注册 / 登录
  - 上传简历链接（简化为 URL）
  - 浏览岗位列表（仅 visible=1）
  - 查看岗位详情并投递
  - 查看自己的投递记录和审核状态
- **企业**
  - 发布 / 编辑岗位
  - 查看本企业岗位列表
  - 查看本企业岗位的所有投递学生，下载 / 查看学生简历链接
- **管理员**
  - 查看全局统计信息（总岗位、总投递、待审核/通过/拒绝数量、各岗位投递人数）
  - 查看全部投递记录并审核（通过 / 拒绝）
  - 查看全部岗位并控制是否显示给学生（visible / hidden）
  - 查看所有学生简历库

### API（示例）

已实现部分与小程序可复用的 REST API，例如：

- `GET /api/jobs`：岗位列表（只返回 visible 岗位）
- `GET /api/jobs/:id`：岗位详情
